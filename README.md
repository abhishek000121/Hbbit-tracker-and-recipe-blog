🍳 FlavorFetch -  Recipe Blog
A modern, visually appealing, and fully responsive Recipe Blog platform built using Bootstrap 5. This project showcases structured content delivery, elegant UI design, and mobile-first development.

🚀 Project Overview
FlavorFetch is designed for food enthusiasts to discover and share recipes. The project focuses on a clean reading experience, category-based filtering, and high-quality visual representation of culinary content.

🛠 Tech Stack
HTML5: Semantic markup for better SEO and recipe indexing.

CSS3: Custom styles for unique typography and brand identity.

Bootstrap 5: Used for the responsive grid system, Navbars, and Recipe Cards.

JavaScript: To handle dynamic search filters and interactive ingredients checklists.

AI Integration: Leveraged Gemini and DeepSeek to generate structured recipe schemas and optimize the responsive layout.

✨ Key Features
Mobile-First Design: Completely responsive layout that works beautifully on smartphones (perfect for use in the kitchen!).

Recipe Cards: Elegant card-based UI featuring high-res images, prep time, and difficulty levels.

Category Filtering: Quick navigation for Breakfast, Lunch, Dinner, and Dessert categories.

Clean Reading Mode: Optimized typography for easy reading of instructions.

Search Functionality: A Bootstrap-powered search bar to find recipes instantly.
